# Changelog

Все важные изменения в проекте mawo-natasha будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-11-01

### Добавлено
- Первый публичный релиз mawo-natasha
- Named Entity Recognition (NER) для русского языка
- Интеграция с Navec embeddings (250K слов, 300 измерений)
- Семантический анализ с векторными представлениями
- Офлайн работа после первой установки
- Опциональная интеграция с mawo-slovnet
- Автоматическая загрузка и кэширование Navec моделей
- Fallback режим с случайными векторами
- Базовые классы: MAWODoc, RealRussianEmbedding, NewsNERTagger
- Менеджер кэша моделей
- Полная документация в README.md
- Тесты интеграции

### Особенности
- Оптимизация для русского языка (+30% точности семантического анализа)
- Производительность: ~5000 токенов/сек, ~2000 embeddings/сек
- Компактный размер: ~60МБ с Navec embeddings
- Поддержка Python 3.10+
