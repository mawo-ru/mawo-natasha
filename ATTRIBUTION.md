## Описание оригинального проекта

Natasha - это библиотека для извлечения именованных сущностей (NER) и семантического анализа русского языка. Проект является ядром экосистемы Natasha и объединяет несколько компонентов:

- **Named Entity Recognition**: Извлечение персон, мест, организаций
- **Navec Embeddings**: Векторные представления русских слов (250K слов, 300 измерений)
- **Семантический анализ**: Работа с векторными представлениями
- **Интеграция**: Использует razdel для токенизации и slovnet для NER

### Основные особенности upstream-проекта:

- Высокая точность распознавания именованных сущностей
- Компактные и эффективные векторные представления
- Интеграция с другими проектами экосистемы Natasha
- Обучение на новостных корпусах
- Быстрая работа на CPU

### Зависимости от других проектов Natasha

Natasha использует следующие проекты из экосистемы:

- **razdel**: Токенизация и сегментация (Copyright (c) 2017 Alexander Kukushkin)
- **slovnet**: Модели NER, морфологии, синтаксиса (Copyright (c) 2017 Alexander Kukushkin)
- **navec**: Векторные представления слов (Copyright (c) 2017 Alexander Kukushkin)

## Улучшения MAWO

Команда MAWO добавила следующие улучшения к upstream-проекту:

- **Автоматическая загрузка данных**: Navec embeddings и модели скачиваются автоматически
- **Offline-first архитектура**: Полная автономная работа после первой загрузки
- **Оптимизация памяти**: ~50МБ для полного функционала (включая embeddings)
- **Улучшенная интеграция**: Seamless интеграция с mawo-razdel и mawo-slovnet
- **Современный инструментарий**: Poetry для управления зависимостями
- **Расширенное тестирование**: Дополнительные тесты и примеры
- **Полная типизация**: Type hints для Python 3.10+
- **Улучшенный API**: Более удобные интерфейсы для работы
